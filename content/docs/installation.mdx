### Installation

Visit: [https://chromewebstore.google.com/detail/read-frog/modkelfkcfjpgbfmnbnllalkiogfofhb?utm_source=official](https://chromewebstore.google.com/detail/read-frog/modkelfkcfjpgbfmnbnllalkiogfofhb?utm_source=official) or search for "Read Frog" in the Google Extension Store.

{/* ![Installation Page](path/to/image) */}

Click the "Add to Chrome" button to install.

### Configuring OpenAI API Key

Click on the plugin button, then click "Settings".

{/* ![Settings Button](path/to/image) */}

On the following page, configure your OpenAI API Key (the current store version doesn't yet support DeepSeek, which will be available in future updates).

{/* ![API Key Configuration](path/to/image) */}

How to get an OpenAI API Key: [https://www.youtube.com/watch?v=SzPE_AE0eEo](https://www.youtube.com/watch?v=SzPE_AE0eEo)

### Start Using

Find a webpage with an article.

{/* ![Example Webpage](path/to/image) */}

Click "Read for Me".

{/* ![Read for Me Button](path/to/image) */}
