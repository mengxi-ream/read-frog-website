# User Guide

<Steps>
<Step>

**Install the extension**

Install Read Frog from the store.

</Step>

<Step>

**Configure API key**

Configure [OpenAI](https://openai.com/api/) or [DeepSeek](https://platform.deepseek.com/) API key.

</Step>

<Step>

**Start using**

Find a webpage with an article and start using.

</Step>
</Steps>

## 1. Installation

Now, you can only install Read Frog from the [Chrome Web Store](https://chromewebstore.google.com/detail/read-frog/modkelfkcfjpgbfmnbnllalkiogfofhb?utm_source=official).

![Installation Page](/tutorial/installation/chrome-web-store.png)

Click the `Add to Chrome` button to install.

<Callout title="Are you using other browsers?">
  In the future, we will support other browsers like Firefox, Edge, and Safari.
</Callout>

## 2. Configuring API Key

Go to the extension page by clicking `Options` button in popup or floating button.

![Options](/tutorial/installation/options.png)

Enter your OpenAI or DeepSeek API key.

### How to get an OpenAI API Key

You can check the [Official Guide](https://help.openai.com/en/articles/4936850-where-do-i-find-my-openai-api-key) or watch the video below.

<iframe
  src="https://www.youtube-nocookie.com/embed/SzPE_AE0eEo"
  className="w-full aspect-video"
  allowFullScreen
/>

### How to get a DeepSeek API Key

You can check the [Official Guide](https://api-docs.deepseek.com/).

## 3. Start Using

Find a webpage with an article.

![Article](/tutorial/installation/start-reading.png)

Click `Read` Button, then wait for a while, you will see the deep explanation of the article.

![Explanation](/tutorial/get-started/interface.png)
