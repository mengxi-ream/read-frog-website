## 1. Installation

Now, you can only install Read Frog from the [Chrome Web Store](https://chromewebstore.google.com/detail/read-frog/modkelfkcfjpgbfmnbnllalkiogfofhb?utm_source=official).

![Installation Page](/tutorial/installation/chrome-web-store.png)

Click the `Add to Chrome` button to install.

<Callout title="Are you using other browsers?">
  In the future, we will support other browsers like Firefox, Edge, and Safari.
</Callout>

## 2. Configuring API Key

Go to the extension page by clicking `Options` button in popup or floating button.

![Options](/tutorial/installation/options.png)

Enter your OpenAI or DeepSeek API key.

### How to get an OpenAI API Key

How to get an OpenAI API Key: [https://www.youtube.com/watch?v=SzPE_AE0eEo](https://www.youtube.com/watch?v=SzPE_AE0eEo)
