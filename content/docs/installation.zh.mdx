## 使用指南

<Steps>
<Step>

**安装扩展**

从商店安装陪读蛙。

</Step>

<Step>

**配置 API key**

配置 [OpenAI](https://openai.com/api/) 或 [DeepSeek](https://platform.deepseek.com/) API key。

</Step>

<Step>

**开始使用**

找到一篇包含文章内容的网页并开始使用。

</Step>
</Steps>

### 1. 安装

目前，陪读蛙仅在 [Chrome Web Store](https://chromewebstore.google.com/detail/read-frog/modkelfkcfjpgbfmnbnllalkiogfofhb?utm_source=official) 上提供。

![安装页面](/tutorial/installation/chrome-web-store.png)

点击 `Add to Chrome` 按钮进行安装。

<Callout title="你使用其他浏览器吗？">
  未来，我们将支持 Firefox、Edge 和 Safari 等其他浏览器。
</Callout>

### 2. 配置 API Key

通过点击弹出窗口或浮动按钮中的 `选项` 按钮进入扩展设置。

![选项](/tutorial/installation/options.png)

输入你的 OpenAI 或 DeepSeek API key。

#### 如何获取 OpenAI API Key

你可以查看[官方指南](https://help.openai.com/en/articles/4936850-where-do-i-find-my-openai-api-key)或观看下面的视频。

<iframe
  src="https://www.youtube-nocookie.com/embed/SzPE_AE0eEo"
  className="w-full aspect-video"
  allowFullScreen
/>

#### 如何获取 DeepSeek API Key

你可以查看[官方指南](https://api-docs.deepseek.com/)。

### 3. 开始使用

找到一篇包含文章内容的网页。

![文章](/tutorial/installation/start-reading.png)

点击 `阅读` 按钮，等待文章详细分析结果出现。

![解释](/tutorial/get-started/interface.png)
